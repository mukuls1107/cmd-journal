Command Line Journal Application Project Outline

This is a beginner-friendly guide to help you create a simple command-line journal application in Python. The program allows users to add, view, search, and delete journal entries.

1. Introduction

The goal of this project is to practice basic programming concepts such as handling lists, dictionaries, loops, and user input. You'll be creating a simple journal application that allows users to interact with their journal.

2. Program Setup

The program will store journal entries in a list called journal. Each entry will be a dictionary with the following information:

  • ID: A unique identifier for the entry.
  • date: The date the entry was created.
  • title: A title for the journal entry.
  • content: The body text of the journal entry.

3. Functions and Their Purpose

3.1 Add Entry

Function Name: add_entry()

Purpose: This function will allow the user to add a new journal entry.

Steps:

  • Ask the user to input the date. If no date is entered, use today's date.
  • Prompt the user for a unique entry ID, a title, and the content of the entry.
  • Store the entry as a dictionary with keys ID, date, title, and content.
  • Append the new entry to the journal list.
  • Confirm that the entry was added.

3.2 View Entries

Function Name: view_entries()

Purpose: This function displays all journal entries stored in the program.

Steps:

  • Check if there are any journal entries in the list.
  • If there are entries, print each entry's ID, title, date, and content.
  • If the journal is empty, inform the user that no entries exist.

3.3 Search Entries

Function Name: search_entries()

Purpose: This function allows the user to search for an entry by its unique ID.

Steps:

  • Ask the user to enter an entry ID.
  • Search through the journal list for the entry with the matching ID.
  • If the entry is found, display its date, title, and content.
  • If the entry is not found, inform the user that the entry doesn't exist.

3.4 Delete Entry

Function Name: delete_entry()

Purpose: This function allows the user to delete a journal entry by its ID.

Steps:

  • Display all journal entries using the view_entries() function.
  • Ask the user for the ID of the entry they wish to delete.
  • Search for the entry by ID and delete it from the journal list.
  • Confirm that the entry was deleted.
  • If the entry ID is invalid, inform the user.

4. Main Program Loop

Function Name: main()

Purpose: This function will run the main menu, allowing the user to select different actions.

Steps:

  • Display a menu with the following options:
    • 1: Add a new entry.
    • 2: View all entries.
    • 3: Search for an entry by ID.
    • 4: Delete an entry by ID.
    • 5: Exit the program.
  • Take the user's input to choose an option.
  • Call the appropriate function based on the user's choice.
  • If the user selects option 5, exit the program with a goodbye message.
  • Repeat the menu until the user chooses to exit.

5. How the Program Works

The program stores journal entries in the journal list. Each entry is a dictionary, which stores the ID, date, title, and content of the journal entry.

Here's an example of how the program works:

  • Adding Entries: Use add_entry() to add a new journal entry.
  • Viewing Entries: Use view_entries() to see all stored entries.
  • Searching Entries: Use search_entries() to search for a specific entry by its ID.
  • Deleting Entries: Use delete_entry() to remove an entry by its ID.

Step 1: Select option 1: Add Entry

Step 3: Enter an ID, title, and content for the entry
Step 4: Confirm that the entry was added

7. Conclusion

This project helps you practice basic input/output, working with lists and dictionaries, and implementing simple functions in Python. Once completed, you can enhance it by adding features like saving the journal entries to a file or implementing additional search options.
